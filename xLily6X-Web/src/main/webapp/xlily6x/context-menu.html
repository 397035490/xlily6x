右键菜单

控件说明

该控件用于需要使用邮件菜单进行操作管理的场景。使用该控件后，当鼠标移至 Target 上时，右键单击，将出现一个关联菜单，用户可根据自己需求设置菜单中的菜单 项，以及单击后的处理事件。

调用实例

Html 代码

界面有div元素，指定 id值


    <div id="cmDemo1" style="width: 400px;height: 200px;background-color: #b4d5c1;border-radius: 12px;text-align: center;padding-top: 60px">
        <h1>右键 Click Me ！  </h1>
    </div>

通过jQuery对象即可调用菜单控件方法

            
    // 初始化 右键环境菜单
    $("#cmDemo1").xLily6X($.widget.ContextMenu,options);

options 对象示例

            
    /**
     * 环境菜单 选项
     */
    var options = {
        // 菜单项
        items:[
            {id:"saveBtn",text:"save1",click:function (obj,env) {
                    console.log(obj);
                    alert("saveBtn is click");
                }
            },
            {id:"save2Btn",text:"save2",click:
                // 单机事件
                function (obj,env) {alert("save2Btn is click");}},
            {id:"save3Btn",text:"save3",click:function (obj,env) {alert("save3Btn is click");}}
        ],
        auto_height:true,
        //背景颜色
        background_color:"#fbf8ff"
    }

options 参数说明

每一个对象都会有一个默认原型，当缺省时，会根据原型中的默认值进行默认加载

名称 类型 默认值 描述
items object "" items 参数用来指定菜单项属性，指定菜单项的ID ，显示的文本，和单机时触发的事件等等，具体可参见下面 items 参数说明
width int 100 菜单的整体宽度，单位 px
height int 100 菜单的整体高度，单位 px ,如需指定高度，请将auto_height 设为false
auto_height boolean true 是否自动设置高度，为true时，设置 height 值无效
background_color string #fbf8ff 背景颜色 默认米白色，如需指定颜色，background_color 设为false
background_color boolean false 自定义设置背景颜色，默认为 false

菜单 options 对象原型


    /*
     *  菜单Options 原型 默认值
     */
    MenuOptions:{
        // 宽度 单位 px
        width:100,
        // 高度 单位 px
        height: 100,
        // 是否自动设置高度，为true时，设置 height 值无效
        auto_height:true,
        // 背景颜色
        background_color:"#fbf8ff",
        // 是否自定义背景颜色，为true时，backgroud_color 设置 无效
        custom_background_color:false,
        items:[
            {id:"menu_item_edit",text:"edit this",click:function (obj,env) {
                console.log(obj);
            }},
            {id:"menu_item_delete",text:"delete this",click:function (obj,env) {
                console.log(obj);
            }}
        ]
    },

item 参数说明

名称 类型 默认值 描述
id string "" 菜单按钮的ID值，可通过这个ID值获取到按钮对象，
text string "" 菜单按钮显示的文本信息
click function 菜单按钮的单机处理事件，如需更多事件，可根据ID获取到对象后自行扩展
名称 类型 默认值 描述
background_color boolean false 自定义设置背景颜色，默认为 false
