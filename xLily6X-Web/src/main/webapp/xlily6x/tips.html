帮助提示框

控件说明

使用该控件后，当鼠标移至 Target 上时，弹出一个帮助提示框，用来对鼠标目标对象进行说明或描述

调用实例

调用实例

Html 代码

效果演示

默认提示框

请将鼠标移至button上

html 代码


    <input type="button" class="xlily6x-button" id="buttond0" value="Tips Button" />

js 代码



    // tips 参数
    var options = {
        font_size:"10px",
        location:"UpRight",
        text:"This is lily tips ! Thanks for you used xlily6x UI !"
    }
    // 控件名称 Tips ， 可通过内置控件名称对象获取 $.widget.Tips
    $("#buttond0").xLily6X($.widget.Tips,options);

不同位置弹出

提示框显示位置

右上 UpRight

右下 LowerRight

左上 UpLeft

左下 LowerLeft

html 代码



    <input type="button"  class="xlily6x-button" id="button1" value="default"/>
    <input type="button"  class="xlily6x-button" id="button2" value="LowerRight" />
    <input type="button"  class="xlily6x-button" id="button3" value="UpLeft" />
    <input type="button"  class="xlily6x-button" id="button4" value="LowerLeft" />

js 代码



    //默认位置 右上
    $("#button1").xLily6X($.widget.Tips,{text:"default location"})
    //右下
    $("#button2").xLily6X($.widget.Tips,{text:"location LowerRight!",location:"LowerRight"})
    //左上
    $("#button3").xLily6X($.widget.Tips,{text:"location UpLeft !",location:"UpLeft"})
    //左下
    $("#button4").xLily6X($.widget.Tips,{text:"location LowerLeft ",location:"LowerLeft"})

options 参数说明

名称 类型 默认值 描述
font_size string 10px 字体大小
location string UpRight 提示框出现的方位
text string 提示框显示的文本内容

注意事项

提示框位置是以鼠标第一个触碰到目标上的点为基准来确定方向， 目标范围内移动鼠标不会改变提示框位置。
