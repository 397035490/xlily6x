快速入门

添加 X-Lily6x 支持

X-Lily6x 扩展于 jQuery ， 所以在引入js 时，需要先引用 jQuery 版本在1.8以上都可以

引入 X-Lily6x js文件


    <script src="../resources/js/jquery-2.1.1.js" ></script>
    <script src="../resources/js/xlily6x-ui-1.0.0.js" ></script>

引入css样式


    <link rel="stylesheet" type="text/css" href="../resources/css/xlily6x-ui-1.0.1.css"/>

使用 X-Lily6x 控件

X-Lily6x 的所有控件使用方式都一样，通过jQuery 对象即可直接调用


    $("#const").xLily6X("widgetName",options);

xLily6X("widgetName",options) 参数说明

名称 类型 默认值 描述
widgetName string "" 在 X-Lily6x 的世界中，所有的控件都有一个名字,这个参数表示需要加载的控件名 控件名可通过 $.widget 对象获取
options object 加载控件所需要的一些初始化配置信息,不同的控件有不同options，具体控件options请看对应控件参数说明
