静态 Data 数据加载

这个下拉框的数据来源于 js 中定义的data 对象

html 代码


    <div id="comboxDemoD1"></div>

js 代码


    // 静态 data 数据
    var data = [{"name":"李云龙","code":"liyunlong"},{"name":"赵小飞","code":"zhaoxiaofei"},{"name":"关云长","code":"guanyunchang"}];

    // 静态 data 数据加载下拉框参数
    var options = {
        value:"code",
        text:"name",
        remote:false,
        data:data,
        onchange:function (obj,args) {
            console.log(obj);
            console.log(args);
            console.log("this comboxDemo1 value changed ");
            // obj 下拉框对象
            alert($(obj).find("option:selected").text()+" : "+$(obj).val());
        }
    };

    $("#comboxDemoD1").xLily6X($.widget.Combox,options)

options 参数说明

名称 类型 默认值 描述
url string 远程请求数据加载地址
data array 下拉框数据源（若非远程请求url加载数据，此属性不可为空）
parameters object 远程请求参数
eg:
{name:"张三丰",status:1} 在请求url地址时，会携带这两个参数
value string 必填项，下拉框选项的值（属性名）
eg:
{"name":"张三","code":"zhangsan"} 将 "code":"zhangsan" 设置为选项的value值 value:"code"
text string 必填项，下拉框选项显示的文本（属性名）
eg:
{"name":"张三","code":"zhangsan"} 将 "name":"张三" 设置为选项的text文本 text:"name"
remote boolean true 是否为远程请求获取数据
dataType string json 远程请求返回的数据类型，目前只支持json格式
method string 远程请求方式，支持 post和 get方式
async boolean true 是否采用异步加载数据，默认为异步加载，false为同步加载
width string 150px 下拉框宽度，默认为150px
onchange function 当下拉框值变动时，触发的事件。 参数 （obj,args）
obj：下拉框对象
args：事件触发参数数组
