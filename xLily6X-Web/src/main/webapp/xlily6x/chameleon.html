表格隔行变色

控件说明

该控件主要用于表格中，其作用是美化表格行，使用后，会把表格隔行渲染自己喜欢的颜色

当鼠标移至 Target 上时，鼠标所在的行将高亮显示出来

调用实例

调用实例

js 代码


    var options = {
        oddClass:"lily_odd",
        evenClass:"lily_even",
        hoverClass:"lily_hover"
    }


    // 可传入Options 参数，
    $("#cnTable").xLily6X($.widget.options);

名称 类型 默认值 描述
font_size string 10px 字体大小
location string UpRight 提示框出现的方位
text string 提示框显示的文本内容
text string 提示框显示的文本内容

options 参数说明

名称 类型 默认值 描述
oddClass string lily_odd 基数行的类样式，可写一个样式，传入
evenClass string lily_even 偶数行类样式
hoverClass string lily_hover 鼠标移至行时高亮显示类样式
